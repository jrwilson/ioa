ioa++

The ioa++ framework is an implementation of the I/O automata formal model in C++ for POSIX environments.

Overview

The I/O automata model has been widely used in the design and verification of asynchronous and concurrent systems. The goal of ioa++ is to allow programmers to develop asynchronous and concurrent, e.g., distributed, systems directly using I/O automata. The strong connection to the formal model helps when developing and debugging a system since the techniques for reasoning about I/O automata can be applied directly to the source code.

Download

  • ioa-0.01.tar.gz (html) (pdf)

You can also clone the project with Git by running: 

$ git clone git://github.com/jrwilson/ioa

Install

$ tar xzf ioa-version.tar.gz
$ cd ioa-version
$ ./configure
$ make
(optional) 
$ make check
As root: 
# make install
# ldconfig
